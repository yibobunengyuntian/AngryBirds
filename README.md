## 项目介绍：

该项目是一个基于Qt框架开发的绘图工具，主要功能特点包括： 采用QPainter实现画板功能，支持铅笔、橡皮、填充、文本、图片插入等绘图工具 采用双图层设计（背景层+画布层）和当前图元绘制机制 支持颜色/图片背景切换、透明度调整和画布尺寸修改 提供撤销/重做功能（基于QUndoStack实现） 可保存为专有格式(.drawing)或导出为常见图片格式
